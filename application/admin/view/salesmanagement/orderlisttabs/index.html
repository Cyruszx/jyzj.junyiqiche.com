销售订单中心&销售签单流程
  1. 请根据客户实际购车方案选择对应购车方式

  2. 如你的实际销售方案与系统发布的方案不一致，请向方案人员确认沟通，添加专属定制方案。
    以租代购（新车）、全款（新车）方案 由产品经理发布，纯租、全款（二手车）、以租代购（二手车）由租车 车管发布，请选择合适的方案发布人员沟通确定方案

  3. 请事先尽量在线下通知客户需要准备齐全购/租的所有资料，包括审核资料、提车资料等
  4. 如你执意要添加、销售与系统发布的不一样的方案，该条订单可能会被永久删除，你需要重新录入销售订单才可以继续销售，请知悉
